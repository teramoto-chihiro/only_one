# README
## 開発言語
- Ruby 2.6.5
- Ruby on Rails 5.2.5

## 就業Termの技術
- お気に入り機能
- devise

## カリキュラム外の技術
- active-storage
## 実行手順
```
$ git clone git@github.com:teramoto-chihiro/only_one.git
$ cd only_one
$ bundle install
$ rails db:create db:migrate
$ rails s
```
